For a time being, the number of dimensions is set to 3 (for plotting
purposes). After running MDS in the entire data, the result will be use
to analyze in this 3 dimesional space the differences in object
exploration conditions (as well as resting and non exploring periods)
and different objects configurations (1, 3 or 6 posibilities depending
on the conditions).

This notebook does not run the MDS (for that check multiscaling.py
rutine), but it loads it results and plots it in different coditions.
